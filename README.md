# SafeRouteAI

Real-Time Risk-Aware Navigation System using Machine Learning.

## 📌 Overview
SafeRouteAI predicts risk levels of locations based on crime data and suggests the safest route between two points.

## 🚀 Features
- Risk prediction using Random Forest
- Multiple route generation
- Safest route selection
- Map visualization using Folium

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Folium

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python app.py

## 📊 Output
- Displays routes on map
- Highlights safest route
- Shows risk levels

## 📁 Dataset
Chicago Crime Dataset
